# Page
Proyecto Web Personal – Desarrollo con IA

Objetivo

Dada la necesidad de documentar mi carrera profesional, he decidido emprender un proyecto ligeramente fuera de mi área de estudio. El objetivo principal es desarrollar y mejorar mis habilidades lógicas, deductivas y de programación, al tiempo que creo una herramienta funcional que documente mi experiencia profesional y académica.

Premisa del Proyecto

Hace unos meses, inicié la creación de una página web utilizando inteligencias artificiales:

	1.	ChatGPT – Generación inicial de la estructura y contenido.
	2.	Grok – Revisión y corrección del código generado.
	3.	Bolt – Pulido final y optimización de la versión casi completa de la página.

El proyecto sirve tanto como experimento de desarrollo asistido por IA como plataforma para mostrar mis logros y proyectos.

Alcance

La página web incluye:
	•	Sección de Acerca de mí
	•	Portafolio de Proyectos y Publicaciones
	•	Página de Currículum Vitae imprimible y actualizable
	•	Diseño minimalista con animaciones sutiles
	•	Estructura modular que permite fácil expansión futura

Tecnologías y Herramientas
	•	Lenguajes: HTML, CSS, JavaScript (prototipo original)
	•	Backend: Python (Flask, con posible migración a Django si se requiere mayor robustez)
	•	Librerías: BeautifulSoup, Scrapy, Plotly
	•	IA usadas: ChatGPT, Grok, Bolt
	•	Entorno de desarrollo: Entorno virtual Python (.venv), VS Code
